# Content

Write and store source text here. Markdown files belong in the folder that matches the kind of post.

- `poems/` for poems.
- `writing/` for blog posts, project notes, reflections, and guides.
- `visuals/photos/` for photo captions and context.
- `visuals/illustrations/` for illustration notes, story art, and references.

Keep image files themselves in `media/` and link to them from Markdown when needed.
